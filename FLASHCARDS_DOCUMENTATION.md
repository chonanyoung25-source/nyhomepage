# 플래시카드 (Flashcards) 기능 구현 완료

## 구현된 기능

### 1. 양면 구성 ✅
- **앞면**: 영어 단어 + 품사 표시
  - 대형 타이포그래피로 단어 강조
  - 품사 정보 (n., v., adj. 등)
  - "Tap to flip" 안내
  
- **뒷면**: 한글 뜻 + 예문
  - 한글 의미 표시
  - 실제 사용 예문 (영어)
  - "Do you know this word?" 프롬프트

### 2. 학습 모드 (Swipe 기능) ✅
- **Know It! (Swipe Right)**: 아는 단어로 분류
  - 초록색 그라디언트 버튼
  - localStorage에 자동 저장
  - 통계에 실시간 반영

- **Don't Know (Swipe Left)**: 모르는 단어로 분류
  - 빨간색 그라디언트 버튼
  - 복습이 필요한 단어로 표시
  - localStorage에 자동 저장

### 3. 자동 재생 기능 ✅
- **Auto-play 모드**:
  - Play 버튼으로 자동 재생 시작
  - 설정 가능한 속도 (3~10초)
  - 자동으로 카드 뒤집기 + 다음 카드 이동
  - Pause 버튼으로 언제든 중단

### 4. 프리미엄 UI/UX ✅

#### 3D 카드 뒤집기 애니메이션
- **Flip Effect**: 실제 카드를 넘기는 듯한 3D 회전 애니메이션
- **부드러운 전환**: 700ms 트랜지션
- **CSS 3D Transform**: perspective, backface-hidden 활용

#### 스와이프 애니메이션
- 좌측 스와이프: 빨간 계열 (모르는 단어)
- 우측 스와이프: 초록 계열 (아는 단어)
- 300ms fade-out + translate 효과

#### 색상 테마
- **앞면**: Emerald/Green 그라디언트
- **뒷면**: Purple/Pink 그라디언트
- **배경**: Emerald-950 → Black 그라디언트
- **유리형태 효과**: backdrop-blur + 투명도

#### 통계 대시보드
- **Total Cards**: 전체 카드 수 (파란색)
- **Known**: 아는 단어 수 (초록색)
- **Review Needed**: 복습 필요 단어 수 (빨간색)

#### 진행률 표시
- 상단 프로그레스 바
- 현재 위치 표시 (예: 45 / 129)

### 5. 추가 기능 ✅

#### 네비게이션
- **이전/다음 버튼**: 카드 간 자유로운 이동
- **키보드 없이 사용 가능**

#### 설정 패널
- Auto-play 속도 조절 (슬라이더)
- 애니메이션으로 패널 표시/숨김

#### 리셋 기능
- 학습 진행 상황 초기화
- 확인 대화상자로 실수 방지
- localStorage 데이터 완전 삭제

## 사용 방법

### 1. 플래시카드 시작
- Vocabulary 페이지에서 "FLASHCARDS" 카드 클릭
- 또는 직접 `/flashcards` 로 이동

### 2. 카드 학습
1. **카드 보기**: 앞면에 영어 단어 표시
2. **카드 뒤집기**: 카드를 클릭하여 뜻 확인
3. **분류하기**: 
   - "Know It!" 버튼 → 아는 단어로 저장
   - "Don't Know" 버튼 → 복습 필요 단어로 저장

### 3. 자동 재생 사용
1. "Auto-play" 버튼 클릭
2. 설정에서 속도 조절 (기본 5초)
3. 자동으로 카드가 뒤집히고 넘어감
4. "Pause" 버튼으로 중단

### 4. 학습 관리
- **진행 상황 확인**: 상단 통계 카드에서 실시간 확인
- **이전 카드 복습**: 이전 버튼으로 되돌아가기
- **처음부터 다시**: Reset 버튼으로 초기화

## 데이터 저장

- **localStorage 사용**: 학습 진행 상황 영구 저장
- **저장 항목**:
  - 단어 ID
  - 단어 텍스트
  - 상태 (known/unknown)
  - 마지막 복습 시간
  - 복습 횟수

## 기술적 특징

### 3D 카드 애니메이션
```css
perspective: 1000px;
transform-style: preserve-3d;
backface-visibility: hidden;
transform: rotateY(180deg);
```

### 자동 재생 로직
- `setInterval`을 사용한 타이머 구현
- 카드가 뒤집히지 않았으면 → 뒤집기
- 카드가 뒤집혔으면 → 다음 카드로 이동

### 반응형 디자인
- 카드 비율: 3:4 (세로형 카드)
- 최대 너비: 2xl (672px)
- 모바일 친화적 버튼 크기

## 학습 효과

### 이미지 연상 학습
- 큰 글씨로 단어 각인
- 색상 변화로 뜻 연상
- 예문으로 맥락 이해

### 능동적 학습
- 스스로 아는지 판단
- 즉각적인 피드백
- 반복 학습 유도

### 자투리 시간 활용
- 자동 재생으로 편리한 학습
- 빠른 속도로 많은 단어 복습
- 언제든 일시정지 가능

## 향후 개선 가능 사항

1. **필터링**: 아는 단어만/모르는 단어만 보기
2. **셔플 모드**: 랜덤 순서로 카드 표시
3. **음성 재생**: 단어 발음 듣기
4. **터치 제스처**: 실제 스와이프 동작 지원
5. **학습 통계**: 일별/주별 학습 기록
6. **북마크**: 특정 단어만 모아서 복습
7. **테마 선택**: 다양한 색상 테마
