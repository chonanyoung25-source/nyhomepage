# 오답 노트 (My Error Note) 기능 구현 완료

## 구현된 기능

### 1. 자동 수집 ✅
- **퀴즈에서 틀린 단어 자동 저장**: Vocabulary Quiz에서 틀린 답을 선택하면 자동으로 오답 노트에 저장됩니다
- **미암기 상태로 저장**: 처음 틀린 단어는 '미암기(unlearned)' 상태로 저장됩니다
- **중복 추적**: 같은 단어를 여러 번 틀리면 틀린 횟수가 누적됩니다

### 2. 복습 주기 알림 (에빙하우스 망각곡선 활용) ✅
- **1일 후**: 첫 번째 복습 (처음 틀린 후 또는 복습에서 틀렸을 때)
- **3일 후**: 두 번째 복습 (1회 연속 정답)
- **7일 후**: 세 번째 복습 (2회 연속 정답)
- **복습 알림**: 메인 대시보드에서 오늘 복습해야 할 단어 개수를 표시

### 3. 마스터 기능 ✅
- **3회 연속 정답**: 복습에서 3번 연속 맞추면 '암기 완료(mastered)' 상태로 이동
- **성취감 부여**: 암기 완료된 단어는 별도 섹션에서 확인 가능
- **진행률 표시**: 각 단어마다 현재 연속 정답 횟수 표시 (예: 2/3)

### 4. UI/UX ✅
- **통계 대시보드**: 
  - 전체 단어 수
  - 미암기 단어 수
  - 학습 중 단어 수
  - 암기 완료 단어 수
  - 오늘 복습 필요한 단어 수
  
- **필터링 기능**:
  - 전체 보기
  - 복습 필요
  - 미암기
  - 학습 중
  - 암기 완료

- **리스트 형태 표시**:
  - 단어와 뜻
  - 틀린 횟수 (X 아이콘으로 시각화)
  - 연속 정답 횟수 (체크 아이콘으로 시각화)
  - 다음 복습 날짜
  - 상태 배지 (색상으로 구분)

- **복습 모드**:
  - 카드 형식으로 단어 제시
  - 정답 보기 버튼
  - 맞혔는지 틀렸는지 선택
  - 진행률 표시

## 사용 방법

### 1. 오답 노트 접근
- Vocabulary 페이지에서 "MY ERROR NOTE" 카드 클릭
- 또는 Quiz 페이지 상단의 "My Error Note" 버튼 클릭
- URL: `/error-note`

### 2. 오답 자동 수집
- Vocabulary Quiz에서 문제를 풀고 틀린 답을 선택
- 자동으로 오답 노트에 저장되며 "오답 노트에 추가되었습니다!" 메시지 표시

### 3. 복습하기
- 오답 노트 페이지에서 "오늘의 복습 시작하기" 버튼 클릭 (복습할 단어가 있을 때만 표시)
- 카드 형식으로 단어가 제시됨
- "정답 보기" 클릭하여 뜻 확인
- "맞혔어요" 또는 "틀렸어요" 선택하여 진행

### 4. 단어 관리
- 필터를 사용하여 상태별로 단어 보기
- 휴지통 아이콘으로 단어 삭제 가능
- 각 단어의 통계 정보 확인 (틀린 횟수, 연속 정답, 다음 복습일)

## 데이터 저장

- **localStorage 사용**: 브라우저의 localStorage에 데이터 저장
- **영구 보관**: 브라우저를 닫아도 데이터 유지
- **자동 동기화**: 퀴즈를 풀 때마다 자동으로 업데이트

## 디자인 특징

- **다크 그라디언트 테마**: Purple/Pink 그라디언트로 프리미엄 느낌
- **유리형태 효과**: Backdrop blur와 투명도를 활용한 모던한 디자인
- **애니메이션**: 호버 효과, 페이드 인 애니메이션으로 생동감 부여
- **색상 코드**:
  - 미암기: 빨간색 (Red)
  - 학습 중: 노란색 (Yellow)
  - 암기 완료: 초록색 (Green)
  - 복습 필요: 주황색 (Orange)

## 향후 개선 가능 사항

1. **통계 차트**: 학습 진행률을 그래프로 시각화
2. **복습 알림**: 브라우저 알림으로 복습 시간 리마인더
3. **내보내기/가져오기**: 오답 노트 데이터를 파일로 저장/불러오기
4. **음성 재생**: 단어 발음 듣기 기능
5. **게임화**: 연속 복습 스트릭, 배지 시스템 등
